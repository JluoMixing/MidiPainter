# MidiPainter 0.2.0

## Highlights

- Drop PNG, JPG, JPEG, WEBP, or BMP files directly onto the input preview.
- Piano-roll previews now stay in the application and no longer write preview PNG or temporary MIDI files.
- MIDI export opens a save dialog in the input image's folder.
- Changing Aspect, pitch range, total beats, or Detail refreshes the preview automatically.

- `MidiPainter-0.2.0-win64-portable.zip`
- `MidiPainter-0.2.0-win64-setup.exe`
